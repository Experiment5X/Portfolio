Rochester Software AssociatesJan-Aug 2017

Software Engineering Co-Op

I worked on a team with 2 other people writing software for coordinating communication with several different printers. The main feature of the software is being able to print to almost any printer out there. A lot of the work involved reverse engineering printer network traffic using Wireshark to understand how to talk to new printers. During my time there I fully implemented IPP which added support for a whole family of printers. I also implemented a couple more specific protocols to add support for legacy printers.

In addition to maintaining printer communication software, my team worked on a desktop application for creating documents to print, similar to MS Word. It had a feature for removing hole punch marks from scanned pages but it only worked for black and white documents and did a pretty bad job of removing them. I created my own algorithm using OpenCV and a CNN I trained myself. OpenCV would detect regions that were likely to contain a hole punch using its blob detector and then the CNN would classify the region as a hole or non-hole. To remove the mark I came up with an algorithm to paint over the region with a color similar to the surrounding area. This feature exists in their application ReadyPrint today.
