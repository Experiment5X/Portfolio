Notable Classes

Deep Learning for Computer Vision

The lectures for this class started off discussing basic neural networks, and then quickly advanced to talking about CNNs before getting to state of the art techniques. For this class I read 5-10 papers on recent techniques used. The lectures also touched on RNNs, LSTMs, auto-encoders, and neural turing machines. There were 3 assignments for the class:

  1. Implementing a neural network from scratch, and deriving the equation for back propagation
  2. Simple CNN classifier, creating adversarial example using Keras
  3. Reinforcement learning with MDP, RNN with handpicked weights

Pythonnumpy KerasCNNs
Web Security

In this class I learned all about the basics of web security including XSS, CSRF, click jacking, CSP, SOP, and SQL injection. This class had 6 well crafted labs in which I did things such as set up a load balancer, create an HTTP client, and pentest a website the professor made. The project for the class is where I learned the most. I made a website like twitter which was distributed across 4 different web services and involved using MySQL for storing user credentials and ElasticSearch for the tweets. In addition to building the system I had to make sure it was secure to prevent things like CSRF, XSS, password cracking, and SQL injection.

DockerDjangoRailsSpringNodeElasticSearch
Authentication

This class focused on the theoretical aspects of security and the mathematical models which are the basis for modern systems. We also touched on authentication schemes such as OAuth, and Kerberos. For the final project, I researched CAPTCHAs and trained a CNN that could solve them with a 45% success rate. That project is on GitHub.

PythonMachine LearningCNN
Real Time and Embedded Systems

The first half of the semester focused on writing bare metal applications. In teams of 2 we made a primitive signal analyzer and a basic servo motor controller. The second half of the semester involved writing applications using FreeRTOS so that we could make use of threads. We built a bank simulation, and then a threaded servo controller. To finish off the semester we built a game where the computer controls one servo motor and the player controls the other by tilting the STM32 board. The player tries to move their servo to match the computer's servo in a set amount of time.

In this class I learned about signals, basic circuits, and how to read the STM32 manual to interact with all of the peripherals that the board offers. I learned how to use the timers, signal receiver, signal generator, UART, and gyroscope.

CSignalsSTM32 FreeRTOS
Engineering of Software Subsystems

I learned about software architecture and design patterns in object oriented systems. Each class we would learn about a different design pattern and do an exercise related to it. The first group project for the class was to design and implement a desktop application for managing stocks using design patterns such as composite, and command. The second group project was to take an existing desktop application that was written without any design patterns in mind and refactor a part of it to be much cleaner using a design pattern. We chose to use the state pattern to represent a frame as part of the scoring system.

JavaOOP Design
Stellar Astrophysics

This class was all about understanding the physics of stars such as how they're formed, how they produce energy and how they evolve over time. I worked on a project with another classmate where we used data from the Gaia telescope to create visualizations showing the distribution of star types in different regions of our galaxy.
