Hello world, I'm Adam Spindler

I have been programming since 8th grade and have worked on a variety of different projects. I’ve reverse engineered kernel byte code, implemented network protocols, created deep learning models, and done full stack web development. I love learning about programming and computers so I can build new and exciting things.

Interests

Machine Learning

Machine learning was first introduced to me when my friend asked me to follow Andrew Ng's machine learning class on Coursera. I watched the lectures and did the assignments which gave me a basic foundation for the field. I thought it was super cool that math and programming together could be used to solve computer vision problems that seem very difficult. The next summer while working at RSA I developed an algorithm involving a CNN to detect and remove hole punch marks from scanned documents. That fall I took a class on deep learning for computer vision in which I derived and wrote a neural network from scratch. I've worked on a few different personal projects involving machine learning including my Garbage Full Detector and Deep Feature Visualization.

Security

I first got interested in computer security when I was in middle school and wanted to hack my xbox games. I started by writing simple programs to edit the binary save files for Halo 3. I eventually learned about file systems and how to reverse engineer assembly code to understand the proprietary formats used on the Xbox 360 to write Velocity. At RIT I minored in computer security where I learned about web security, and how to write secure applications.

Web Development

I'm interested in developing applications on the web because the platform is so huge. A new web application can be used by anyone on any device without having to install any additional software. Web applications also have a lot of components to them which I think make them more interesting. It's cool to see all the pieces work together to make it actually do something. I've done web development professionally at both EagleView and datto.

Teaching

I love teaching other people about programming and trying to convince them to get into it. I have successfully convinced a couple of my friends and have happily tutored one of them for the past few semesters. Before I got to college I made 200 C# tutorials on YouTube for thenewboston's channel which collectively reached over 2 million views. I also made 87 C tutorials for my own channel and they gained over 1 million total views. Creating both of these series taught me a ton about each of the languages.
